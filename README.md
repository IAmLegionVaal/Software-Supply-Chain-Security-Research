# Software Supply Chain Security Research

A defensive research repository for studying software dependency, build, provenance, and repository security controls.

## Research areas

- Dependency inventory and age
- Lock-file and checksum coverage
- Build workflow permissions
- Release provenance and artifact integrity
- Secret-scanning and branch-protection readiness
- SBOM availability
- Third-party action pinning

## Main tool

```powershell
powershell.exe -ExecutionPolicy Bypass -File .\Software_Supply_Chain_Security_Research.ps1 -RepositoryPath C:\Path\To\Project
```

## Output

CSV findings, JSON evidence, and an HTML research report.

## Safety

Read-only local repository analysis. No files, workflows, or dependencies are changed.
